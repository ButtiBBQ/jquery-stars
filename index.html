jQuery Stars cursor animation

This is magic jquery plugin that add "magic" to any element on your site. Very easy to use

					
$('#example-1').jstars({
image_path: 'images', // folder with magic image
style: 'white',       // optional, magic style, default = white
frequency: 5          // optional, from 1 to 19
});

container with white stars
